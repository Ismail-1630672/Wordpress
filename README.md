# Wordpress Assignment

# Objective:
The main objective is to deploy wordpress to AWS via an EC2 instance utilising Terraform and Docker

# Terraform:
Will be used to build the infrastructure such as subnets, security groups, route tables, EC2 instance etc

# Docker:
Will be used to pull the wordpress image and then run it inside the EC2 instance

# EC2 instance:
Will need to connect to EC2 instance by using SSH via .pem, wordpress will be deployed here.
